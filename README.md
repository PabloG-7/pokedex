# <img src="https://raw.githubusercontent.com/PokeAPI/sprites/master/sprites/pokemon/other/official-artwork/25.png" width="40" height="40"> Pokédex React

Uma Pokédex interativa construída com React e PokéAPI, com design moderno e responsivo para explorar o mundo Pokémon.

![Demonstração da Pokédex](https://raw.githubusercontent.com/PabloG-7/pokedex/refs/heads/main/pokedex-apresentacao.png)

## 🔗 Demo Online

➡️ [Acesse a Pokédex aqui](https://pokedex-eight-drab-61.vercel.app/) ⬅️

## ✨ Funcionalidades Principais

- 🎯 **Listagem paginada** de Pokémon (100 por página)
- ⚡ **Busca instantânea** por nome ou tipo
- 📊 **Detalhes completos**:
  - Tipos elementais
  - Estatísticas (HP, Ataque, Defesa)
  - Habilidades especiais
  - Altura e peso
- 🌈 **UI Atraente**:
  - Gradientes dinâmicos por tipo
  - Cards interativos
  - Animações fluidas
- 📱 **Totalmente responsivo** para todos dispositivos

```bash
npm install
npm start

```
