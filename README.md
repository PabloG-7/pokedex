# Pokédex

Uma Pokédex interativa construída com *React* e PokéAPI, permitindo listar, buscar e visualizar detalhes de Pokémon. O projeto apresenta um design moderno, responsivo e com uma experiência de usuário fluida.

## Demonstração

Acesse a Pokédex online aqui: [Pokédex](https://pokedex-ten-chi.vercel.app/)

## Funcionalidades

- *Listagem de Pokémon*: Exibe uma lista paginada de Pokémon com imagens e nomes (100 Pokémon por vez na página inicial).
- *Busca Dinâmica*: Permite buscar Pokémon por nome, exibindo todos os resultados correspondentes de uma vez.
- *Detalhes do Pokémon*: Mostra informações detalhadas (tipo, altura, peso, habilidades) ao clicar em um Pokémon.
- *Design Responsivo*: Layout adaptável para diferentes tamanhos de tela (desktop, tablet, celular).
- *Estilização Moderna*: Interface com gradiente temático, sombras, animações sutis e cards interativos.

## Tecnologias Utilizadas

- *React*: Biblioteca JavaScript para construção de interfaces dinâmicas.
- *PokéAPI*: API pública para obter dados de Pokémon.
- *Axios*: Para realizar requisições HTTP à PokéAPI.
- *Tailwind CSS*: Framework CSS para estilização rápida e responsiva.
- *GitHub Pages*: Hospedagem do site.
